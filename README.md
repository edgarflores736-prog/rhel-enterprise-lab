# RHEL Enterprise Lab

## About

This repository contains hands-on Red Hat Enterprise Linux (RHEL) administration and cybersecurity labs.

The goal is to demonstrate practical skills for Linux System Administrator, Systems Engineer, and Cybersecurity Engineer positions.

## Technologies

- Red Hat Enterprise Linux 10
- Bash
- Git
- SSH
- Firewalld
- SELinux
- LVM
- Ansible
- Docker
- Splunk
- Wazuh
